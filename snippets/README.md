# Snippets（JS コード断片・ユーティリティ集）

JS Templates を補助する小さなコード断片をまとめたディレクトリです。

- 「書き始め」を楽にする雛形
- よく使うユーティリティ関数
- PC/モバイル両対応のイベント束ね処理 など

Toolkit からは Templates → **Snippets** として一覧表示され、  
選択したスニペットをエディタへワンクリック挿入できます。

---

## 📁 収録例

| ファイル                 | 役割                                      |
| -------------------- | --------------------------------------- |
| `basic-utility.js`   | API ショートカット / ロガー / クエリエスケープなどの基本ユーティリティ |
| `config-template.js` | `CONFIG` ブロックの雛形                         |
| `event-bundling.js`  | PC/モバイル両対応のイベント束ね処理                    |

※ 実際の収録内容は GitHub 上の一覧を参照してください。

---

## 🔧 使い方

### 1. 直接コピペで使う場合

1. Snippets から対象ファイルを開く
2. 必要な部分だけコピーし、自分のカスタマイズ JS に貼り付ける
3. フィールドコードやロジックをアプリに合わせて調整する

### 2. Toolkit から挿入する場合

1. Toolkit の **Templates タブ** を開く
2. Source を `Snippets` に切り替える
3. 挿入したいスニペットを選択 → 「挿入」ボタン

JS Templates と組み合わせることで、

- 設定用 `CONFIG` の標準化
- ログ出力や小さなヘルパの共通化

など、「Toolkit 流」の書き方を簡単に取り込めます。
