# kintone Customization JS Templates

kintone のレコード画面や一覧画面で使える、汎用的な JavaScript テンプレート集です。  
**すべて1ファイルで完結**しており、自由に改変・再利用が可能です。

---

## 📁 ファイル構成

| ファイル名 | 概要 | 主な用途 |
|-------------|------|----------|
| `field-hide-disabled.js` | 指定したフィールドを非表示または入力不可にする | 権限によって編集制限したい時など |
| `field-validator.js` | 条件式によるフィールドバリデーション。`with`演算子で他フィールド参照も可能 | 保存前に条件エラーを検出したい時 |
| `group-tab-switcher.js` | 「グループフィールド」をタブUIとして切り替え可能にする | 大きなフォームをタブで分けたい時 |
| `index-quick-switch-buttons.js` | 一覧画面のヘッダにナビゲーションボタンを設置 | 一覧の切り替えや外部リンクを簡単に配置 |

---

## 🚀 使い方

1. 各 `.js` ファイルをダウンロード  
2. kintone の **「JavaScript / CSSカスタマイズ」** にアップロード  
3. 対象アプリに適用  

それぞれのファイル上部にある `CONFIG` 定数を編集することで、  
非表示フィールド・条件式・タブ・ボタンなどを自由にカスタマイズできます。

---

## 🧩 共通仕様

- すべて **kintone標準API（kintone.events.on / kintone.app.record / kintone.app.getHeaderSpaceElement）** のみ使用  
- 外部ライブラリ不要  
- デスクトップ環境で動作確認済み（モバイルはコメントアウト行を有効にすれば対応可能）  
- 配布・改変・組み込み自由（商用利用OK）

---

## 💡 開発者メモ

- 4つのスクリプトはいずれも独立して動作します。
- まとめて使う場合は `CONFIG` のフィールドコードが重複しないように注意。
- すべての `.js` は学習目的でも利用可能です（コードコメントは詳細説明付き）。
